## 3. Model Development & Training
**Model Architecture:**  
- **Inception V3 (Google)**  
  - Robust feature extraction using transfer learning
  - Pre-trained on large dataset (ImageNet)  

**Hyperparameter Tuning:**  
- Learning Rate: 0.0001  
- Epochs: 20  
- Batch Size: 16  
- Momentum: 0.9  
- Dropout Rate: 20%  
- L2 Regularization: 0.05  

**Performance Metrics:**  
- Training Accuracy: 79.3%  
- Testing Accuracy: 80%
